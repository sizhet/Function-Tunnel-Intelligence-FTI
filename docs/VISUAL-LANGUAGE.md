# FTI Visual Language

## Green Tunnel

Represents:

- acceptable behavioral manifold,
- structural corridor,
- survivable runtime geometry.

## Blue Trajectory

Represents:

- stable structural homing,
- adaptive flow,
- successful runtime continuation.

## Red Broken Path

Represents:

- rigid exact-control failure,
- brittle navigation collapse.

## CCC Anchors

Represent:

- invariant structural cores,
- homing references,
- attractor points.

## Tunnel Width Gradient

Represents:

- tolerance confidence,
- dynamic acceptance flexibility,
- adaptive robustness.

## Swarm Field Patterns

Represent:

- decentralized structural coordination,
- collective tunnel alignment.
